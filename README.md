# churchtools-api

Tests mit dem Churchtools API-Client von https://github.com/5pm-HDH/churchtools-api

## Installation
- Repository klonen
- Dependencies mit ```composer install``` installieren

## Credentials
Datei 'credentials.inc' anlegen mit folgendem Inhalt:
```
<?php

$url = "CHURCHTOOLS-URL";
$usr = "USER";
$pwd = "PASSWORD";
```

## Skript aufrufen
```php index.php```
